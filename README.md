# Compliance

Section dédiée aux documents de conformité de VoxelsFX Studio.

Ce dossier regroupe :

- Registre des activités de traitement RGPD
- Cartographie des contrôles de sécurité ISO 27001 (SoA raccourcie à une dizaine de contrôles)

Ces documents permettent de structurer la conformité réglementaire du studio en identifiant les traitements de données personnelles et en définissant les mesures de sécurité applicables afin de garantir le respect du RGPD et la protection des informations sensibles.
